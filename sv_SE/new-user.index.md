Välkommen till Firefly III!

För att komma igång, fyll i nödvändiga fält.

* Banknamnet behövs för att ge ditt konto ett bra namn.
* Ditt saldo just nu på kontot. Du kan sätta det till 0 och korrigera det senare.
* För ditt sparkonto så gäller samma sak. Namnet kommer att tas från namnet på din bank, men du kan ändra det senare.

Läs mer om Firefly III i den [officiella dokumentationen](https://docs.firefly-iii.org/).
